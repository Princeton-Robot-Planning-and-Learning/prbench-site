Benchmark / Geometric 2D / Motion2D

Motion2D

Description

A 2D environment where the goal is to reach a target region while avoiding static obstacles.

There may be narrow passages.

The robot has a movable circular base and a retractable arm with a rectangular vacuum end effector. The arm and vacuum do not need to be used in this environment.

Variants

This environment has 6 variant(s).

Motion2D-p0

This variant has no narrow passages (open space).

Motion2D-p1

This variant has 1 narrow passage.

Motion2D-p2

This variant has 2 narrow passages.

Motion2D-p3

This variant has 3 narrow passages.

Motion2D-p4

This variant has 4 narrow passages.

Motion2D-p5

This variant has 5 narrow passages.

References

Narrow passages are a classic challenge in motion planning.
