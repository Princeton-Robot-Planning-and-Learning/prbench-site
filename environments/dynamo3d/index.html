Dynamo3D

Description

A 3D mobile manipulation environment using the TidyBot platform.

The robot has a holonomic mobile base with powered casters and a Kinova Gen3 arm. Scene type: ground with 3 objects. In the 'ground' scene, objects are placed randomly on a flat ground plane.

The robot can control:

  • Base pose (x, y, theta)

  • Arm position (x, y, z)

  • Arm orientation (quaternion)

  • Gripper position (open/close)

Variants

This environment has 2 variant(s).

Dynamo3D-o1

This variant uses the 'ground' scene type with 3 objects.

TidyBot3D-Dynamo3D-namo-o1

This variant uses the 'namo' scene type with 1 object.
